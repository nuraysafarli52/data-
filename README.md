# data-
bir turizm platformundaki kullanıcıların dijital ayak izlerini ve davranışsal verilerini inceleyerek müşteri profilini anlamayı amaçlayan kapsamlı bir Keşifçi Veri Analizi (EDA) projesidir. Proje, seyahat sayfalarındaki etkileşimler, tercih edilen cihazlar ve sosyal onay  gibi faktörlerin müşteri davranışları üzerindeki etkisini analiz etmektedir.
